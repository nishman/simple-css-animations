Simple CSS Animations

Download from GitHub.

This is a pure CSS library for animating HTML elements. Include the CSS file in your project and use these CSS classes to animate various elements when you click, hover, or tab over them. Each item in the "Single Animations" section is labelled with its class name.

NOTES

  1. Due to browser variations, sometimes hover animations for non-clickable items will be janky. One way to fix this is to wrap the element inside an "a" tag. You can leave out all the normal link attributes if you don't want normal link behavior.
  2. In Mac OSX Firefox and Safari, buttons do not stay focused after you click them. You have to hold down the mouse button (or tab through them one at a time) to see the full click animations. Button animations work normally in all major Windows browsers and OSX Chrome.
  3. For better support with older browsers, use with -prefix-free or autoprefixer.
